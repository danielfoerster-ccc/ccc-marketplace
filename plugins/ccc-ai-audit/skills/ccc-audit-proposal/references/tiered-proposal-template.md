## Tier Scoping Guide

### How to decide what goes in each tier:

**Quick Wins (tier 1):**
- Processes with VALUE score 20+ AND Effort score 4-5
- Can be built in under 5 hours each
- Use pre-built templates or simple trigger→action patterns
- Prove value within 2 weeks
- Should feel like an obvious yes at the price point

**Foundation (tier 2):**
- Everything in Quick Wins
- Processes with VALUE score 15+ AND moderate complexity
- May require custom logic, multiple tools, or testing
- 4-8 week timeline allows for proper QA
- The "real" package — price it to be the default choice

**Full Transformation (tier 3):**
- Everything in Foundation
- All remaining valuable processes
- Advanced features: AI agents, complex integrations, custom code
- Includes documentation, SOPs, and ongoing optimization
- For clients who want the complete solution and have the budget

### Pricing psychology:

- Quick Wins should feel like a no-brainer ("we'd be foolish not to do this")
- Foundation should feel like the smart choice ("this is where the real value is")
- Full Transformation should feel aspirational but justified ("if we're going to do this, let's do it right")
- The gap between Quick Wins and Foundation should be bigger than between Foundation and Full Transformation

## ROI Calculation Reference

```
For each tier, calculate:

Annual Time Savings (hours) = Sum of (Process Time × Monthly Frequency × 12 × Automation%)
                              for all processes included in the tier

Annual Cost Savings = Annual Time Savings × Hourly Rate

Implementation Cost = Tier Price + Annual Tool Subscriptions

First-Year ROI = (Annual Cost Savings - Implementation Cost) / Implementation Cost × 100%

3-Year Savings = Annual Cost Savings × 3

Break-Even (months) = Implementation Cost / (Annual Cost Savings / 12)
```

**Hourly rate:** Use the client's actual blended rate if known. If not, ask the consultant to estimate based on the roles involved. Common ranges: $30-50 for support/admin, $50-100 for professional/manager, $100-200 for senior/executive.

## Conversion Tactics Built Into the Template

1. **Three tiers** — the middle one is the target. Research shows people gravitate to the center option.
2. **"Most Popular" label** — social proof signal on Foundation. Guides the decision.
3. **ROI table** — turns the investment into a math problem. Hard to say no to 300%+ ROI.
4. **Specific deliverables** — not "automation services" but "HubSpot → Make onboarding workflow." Specificity builds confidence.
5. **Expiration date** — creates urgency without pressure. 14 days is standard.
6. **Frictionless next step** — "reply to this email." No forms, no portals, no friction.
7. **Building tiers** — each tier includes the previous one. Creates a clear upgrade path.
