## The Three-Phase Acquisition Model

### Phase 1: Network (Warm) — "Get First Clients"
- **Goal:** 1–3 signed clients via personal network
- **Method:** Personal messages to warm contacts (not templates)
- **Target:** ~20 warm agency contacts (Tier 1), ~200 warm professionals (Tier 2)
- **Non-negotiables:**
  - Calendly set up and tested
  - Outreach tracking spreadsheet live
  - Tier 1 list built (agency contacts identified)

### Phase 2: Inbound / Content — "Build Authority"
- **Goal:** Consistent LinkedIn presence; inbound leads start
- **Method:** 3 posts/week on LinkedIn (learnings, client solutions, personal stories)
- **Target:** LinkedIn connections become warm — the Handraiser post is the flagship
- **Non-negotiables:** Consistency beats quality; volume before perfection

### Phase 3: Scale — "Activate Cold Channels"
- **Goal:** Predictable pipeline from cold outreach
- **Method:** One channel mastered (LinkedIn DMs, Cold Email, OR Cold Calling)
- **Rule:** Pick ONE cold channel. Stay with it for 3 months before evaluating.

## The Acquisition Funnel

```
Awareness (LinkedIn posts / workshops)
    ↓
Engagement (likes, comments, DMs)
    ↓
Intent-based outreach (Trigify → filter ICP → DM sequence)
    ↓
Discovery call booked
    ↓
Proposal + close
    ↓
Delivery (CCC)
```

---

## Core Principle: Distribution First

"Go distribution first. Start today."
- Start acquisition channels on Day 1, even before niche is fully decided
- The more uncomfortable the channel, the more effective (cold call > cold email > DM > content)
- Most unicorns grew through ONE main acquisition channel, not five average ones
