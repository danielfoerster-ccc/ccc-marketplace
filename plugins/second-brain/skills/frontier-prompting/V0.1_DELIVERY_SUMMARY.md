## Architecture Decision: Option B (Inside second-brain Plugin)

**Chosen:** Live inside `second-brain` plugin alongside `ccc-mental-model-recipes`

**Reasoning:**
- Both are cognitive-operations skills (Frontier Prompting designs questions; ccc-mental-model-recipes applies recipes to those questions)
- Same family: skill-to-skill composition (recipes skill will reference Frontier Prompting in v0.2)
- Easier to ship together as part of the unified "Third Brain Mastery stack"
- Reduces plugin bloat vs. standalone approach
- Maintains progressive disclosure: SKILL.md ≤400 lines; references loaded as needed
