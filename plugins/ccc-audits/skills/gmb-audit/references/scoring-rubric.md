# GMB Audit Scoring Rubric

*Last updated: 2026-03-10. Review before use if older than 90 days.*

This rubric defines how to score each GMB dimension from 0–100. Apply each criterion and sum the points. Cap at 100.

---

## Dimension 1: Profile Completeness (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Business name set and correct (no keyword stuffing) | 10 | Check name matches real brand name. Penalize if stuffed with keywords (e.g., "Pizza Berlin Mitte Best Pizza"). |
| Primary category set and appropriate | 10 | Must match the core business type. Wrong or too-broad category = 0. |
| 1+ secondary categories set | 5 | Bonus categories that describe additional services. |
| Business description filled out | 10 | Any text = 5 pts. Keyword-rich, full length (~750 chars) = 10 pts. |
| Phone number set | 10 | Present and correct = 10. Absent = 0. |
| Website URL set | 10 | Present = 10. Absent = 0. |
| Street address set | 10 | Required for brick-and-mortar. Absent = 0. |
| Regular hours set for all open days | 10 | Complete and accurate = 10. Partial = 5. Missing = 0. |
| Special/holiday hours updated in last 90 days | 5 | Shows profile is actively maintained. |
| Booking link / reserve button enabled | 5 | If relevant to business type. N/A for businesses without appointments = award full points. |
| Products or Services listed | 10 | Any items listed = 5 pts. Detailed with prices/descriptions = 10 pts. |
| Attributes filled (accessibility, amenities, payment) | 5 | At least 3 relevant attributes set = full points. |

**Total possible: 100 pts**

## Dimension 2: Visual Content / Photos (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Logo photo present | 15 | High-quality logo = 15. Low quality or absent = 0. |
| Cover photo present and high quality | 15 | Professional, on-brand photo = 15. Stock/generic = 5. Absent = 0. |
| Total photo count ≥ 10 | 20 | 20+ photos = 20 pts. 10–19 = 15 pts. 5–9 = 8 pts. 1–4 = 3 pts. 0 = 0. |
| Photo variety (interior, exterior, product/work, team) | 20 | 4+ categories = 20. 3 = 15. 2 = 8. 1 = 3. |
| Recent photo added in last 90 days | 15 | Owner or customer photo added recently = active signal. |
| Photo quality overall | 15 | Bright, professional, on-brand = 15. Mixed = 8. Blurry/irrelevant = 0. |

**Total possible: 100 pts**

## Dimension 3: Reviews & Reputation (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Average rating ≥ 4.5 | 20 | 4.7+ = 20. 4.5–4.69 = 15. 4.0–4.49 = 8. 3.5–3.99 = 3. <3.5 = 0. |
| Total review count | 20 | 100+ = 20. 50–99 = 15. 25–49 = 10. 10–24 = 5. <10 = 2. 0 = 0. |
| Owner response rate (last 10 reviews) | 20 | Responded to 9–10 = 20. 7–8 = 15. 5–6 = 8. 1–4 = 3. 0 = 0. |
| Response quality (personal vs. generic) | 20 | Personalized, mentions reviewer's name/specific feedback = 20. Generic template = 8. No responses = 0. |
| Recent review in last 30 days | 10 | Yes = 10. Last 60–90 days = 5. Older = 0. |
| Positive keyword mentions (quality, service, etc.) | 10 | 3+ consistent themes = 10. 1–2 = 5. None identified = 0. |

**Total possible: 100 pts**

---

## Dimension 4: Posts & Engagement (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Post in last 7 days | 30 | Yes = 30. 8–14 days = 20. 15–30 days = 10. Older = 0. |
| Posts 2+ per month (based on visible history) | 25 | 4+ posts in last 30 days = 25. 2–3 = 18. 1 = 8. 0 = 0. |
| Post includes a photo | 20 | Most recent 3 posts have photos = 20. Some = 10. None = 0. |
| Post includes a CTA (learn more, book, offer, etc.) | 15 | CTA present in most recent post = 15. Some posts = 8. None = 0. |
| Post variety (updates, offers, events) | 10 | 2+ types used = 10. 1 type only = 5. |

**Total possible: 100 pts**

## Dimension 5: Q&A & Community (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Owner has seeded Q&As (pre-answered common questions) | 40 | 3+ owner-seeded Q&As = 40. 1–2 = 20. 0 = 0. |
| All public questions answered | 30 | All answered = 30. Some = 15. Unanswered questions present = 0. |
| Q&A content is relevant and keyword-rich | 20 | Well-written answers that help searchers = 20. Generic = 8. Absent = 0. |
| Q&A updated in last 6 months | 10 | Recent activity = 10. Old/stale = 0. |

**Total possible: 100 pts**

---

## Dimension 6: Local SEO Signals (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Business description contains service keywords | 20 | Core service words (not stuffed) naturally mentioned = 20. Partially = 10. Generic or missing = 0. |
| Business description contains location keyword | 15 | City or neighborhood mentioned = 15. Absent = 0. |
| Primary category precisely matches core service | 20 | Best-match category = 20. Close match = 10. Wrong = 0. |
| NAP matches website (Name / Address / Phone) | 25 | All 3 match exactly = 25. Minor variance = 15. Significant mismatch = 0. |
| Website has LocalBusiness schema markup | 10 | Valid JSON-LD schema present = 10. Absent or broken = 0. (Check via view-source or Google Rich Results Test) |
| Business in correct city/region = Google considers it local | 10 | Confirmed local = 10. Cross-city ambiguity = 5. |

**Total possible: 100 pts**

---

## Dimension 7: Features & Conversion (100 pts)

| Criterion | Points | How to evaluate |
|-----------|--------|-----------------|
| Booking/Appointments button active | 30 | Functional booking button = 30. Not applicable = award 30 pts. Absent when applicable = 0. |
| Messaging/Chat enabled | 20 | Enabled = 20. Disabled = 0. Not applicable = 20. |
| Website link is correct and working | 20 | Links to relevant page (homepage or booking page) = 20. Dead link or wrong page = 0. |
| Products/Services menu populated with prices | 15 | 5+ items with prices = 15. Items without prices = 8. Empty = 0. |
| Social profiles linked (if applicable) | 15 | Instagram/Facebook linked in profile = 15. None = 0. Not applicable = 15. |

**Total possible: 100 pts**

## Competitor Benchmark Quick Reference

When benchmarking, record these values for each competitor:

| Field | What to note |
|-------|-------------|
| Business name | Exact name |
| Rating | X.X / 5 |
| Review count | Total number |
| Photo count | Visible count |
| Has recent posts | Yes/No |
| Owner responds to reviews | Yes/No |
| Booking button | Yes/No |

Calculate competitor averages for: rating, review count, photo count. Compare target business to these averages to identify where they're ahead or behind.

## Score Interpretation Guide

| Score | Label | Interpretation | Urgency |
|-------|-------|----------------|---------|
| 85–100 | Excellent | Profile is well-optimized. Focus on maintenance. | Low |
| 70–84 | Good | Solid foundation with specific gaps. Target amber dimensions. | Medium |
| 55–69 | Needs Work | Visible opportunities being left on the table. | High |
| 40–54 | Poor | Significant optimization needed across multiple areas. | Urgent |
| 0–39 | Critical | Profile is likely hurting more than helping. | Critical |
