## A8: ALLOCATION LOG (Running Record)

Maintain this table perpetually. One row per allocation cycle. Use for monthly review and annual audit.

| Date | Income Balance (Start) | Profit | Owner Comp | Tax | OpEx | Mat&Sub | OpEx Balance (End) | Days Coverage | Notes |
|------|---|---|---|---|---|---|---|---|---|
| 2026-04-15 | €50,000 | €2,500 | €25,000 | €7,500 | €15,000 | — | €15,000 | 26 days | Initial allocation ✓ |
| 2026-05-10 | €18,500 | €925 | €9,250 | €2,775 | €5,550 | — | €20,550 | 29 days | Rhythm test ✓ |
| 2026-06-10 | €22,000 | €1,100 | €11,000 | €3,300 | €6,600 | — | €27,150 | 35 days | OpEx healthy |
| | | | | | | | | | |

**Fields:**
- Date: Allocation execution date
- Income Balance (Start): Amount in Income account before allocation
- Profit/Owner Comp/Tax/OpEx/Mat&Sub: Amounts transferred to each
- OpEx Balance (End): OpEx account balance after all transfers clear
- Days Coverage: (OpEx Balance ÷ weekly expenses) × 7
- Notes: Any issues, adjustments, special events

**Review quarterly:** Look for trends. Is OpEx consistently short? Is Profit growing? Is Owner Comp stable? Adjust percentages accordingly in [[CCC Cashflow — Quarterly Review Cycle]].
